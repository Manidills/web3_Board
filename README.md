## Problems

* Lack of INTERACTIVE web3 protocols dashboards avilable publically. Dune started a major step on that but still users need sql stuffs to get the req data.
* Basic UI to download protocols/NFT analaytics data
* Finding the trending and rare NFT Tokens are hard.
* No simple user interface to extract and download NFT related or wallet related historical and detailed data.
* Lack of publically available place to scan the codebase and get detailed bug reports for smart contracts. (INDEED)
* Personalized IPFS upload's and URL link managements.


## Solution - WEB3 BOARD
```sh
* Provide complete interactive option to explore the web3 protocols with download data option.
* NFT Tracker that able to find the trending NFTS based on collections
* Simple user interface with the feature to explore the NFTs and wallet level analytics.
* Smart contracts bug scanner with detailed report ( include export option )
* Wallet based IPFS data storage and links managements ( All the files are encrypted using AES-256 Algorithm )
* Place to generate unique art's and mint as a NFT in a minute.
```

2. **Storage_Mangement** - Upload and maintain the IPFS data and URL of the File

    1. **Connect your web3 wallet (Metamask)** to WEB3 Board
    2. **Upload** - Upload the collection metadata or any file you wanna maintain. It uses **pinata** service to **upload the data**.
    3. **check** - once you click check TAB its **retrive** the all upload you made to **IPFS** in a table structure with the **IPFS URL link** of the each item with downlaod option.
    
    **Core focus is to make it personalized way, just connect your wallet, check or upload your data's, if you wish to share any IPFS URL , you juts away with one step.**
